# From Earth

讓我們探索地球上的美麗景點。

<img src="./assets/Screenshot 2023-08-14 at 19.30.13.png" width="512">

## 功能

- 探索數千個美麗的地方
- 設為您的桌布
- 加入到您的收藏

還有更多！
