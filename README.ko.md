# 지구에서

지구의 아름다운 곳을 탐험해봅시다.

<img src="./assets/cover-image.ko.png" alt="Cover Image" width="512">

## 특징

- 수천 개의 아름다운 장소 탐험
- 배경화면으로 설정
- 즐겨찾기에 추가

그 외에도 더!
